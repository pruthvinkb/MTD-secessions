## 🛠️ Technologies Used

### Backend (Java)
- **Spring Boot** - Main framework
- **Spring Web** - REST API development
- **Spring Data JPA** - Database operations
- **Maven** - Dependency management
- **Java 17+** - Programming language

### Frontend (React)
- **React 18+** - UI library
- **Vite** - Build tool
- **ESLint** - Code linting
- **CSS3** - Styling

## 🚀 Getting Started

### Prerequisites
- Java 17 or higher
- Node.js 16 or higher
- Maven 3.6+
- Git

### Running Java Projects
```bash
cd [project-folder]
./mvnw spring-boot:run
```

### Running React Projects
```bash
cd [project-folder]
npm install
npm run dev
```
